# RHMS
# Remote Patient Monitoring System (RPMS)

## Overview
This is a basic simulation of an RPMS using Object-Oriented Programming in Java. It includes src folder that contains all the java files with Main.java to run the program:
src/
Administrator.java
Appointment.java
AppointmentManager.java
ChatClient.java
ChatServer.java
Doctor.java
EmailNotification.java
EmergencyAlert.java
Feedback.java
Main.java
MedicalHistory.java
Notifiable.java
NotificationException.java
NotificationService.java
PanicButton.java
Patient.java
Prescription.java
SMSNotification.java
User.java
VideoCall.java
VitalSign.java
VitalsDatabase.java

## How to Run
1. Clone this repo.
2. Run `Main.java` inside your IDE or Codespaces terminal.
